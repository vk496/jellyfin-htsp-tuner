Tvheadend (HTSP)

These settings drive the integrated Live TV service. They are also the fallback connection for "HTSP Tuner" devices until Jellyfin's UI can show per-tuner fields, so set them here even if you add HTSP under Add Tuner Device.

Tvheadend host name or IP.
Used for channel icons and the HTTP streaming fallback.
Only if Tvheadend sits behind a reverse proxy path.
Leave empty for an anonymous Tvheadend.

Streaming

Tvheadend streaming profile passed to subscribe. Default htsp.
Higher wins when tuners are contended. Tvheadend's live default is 100.
Per-channel in-memory buffer cap. When a client falls behind, oldest data is dropped rather than blocking.
Base URL Jellyfin uses to read the plugin's own live stream. Leave empty to use http://127.0.0.1:<port> (correct for a normal single-host install and avoids Jellyfin picking a wrong network interface). Only set this if Jellyfin's transcoder runs on a different host than its API.

Guide & recordings
